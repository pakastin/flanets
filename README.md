# Flanets.io
Free and open source 2D space flight simulator

## Play!
https://flanets.io

## Install dependencies
`npm i`

## Run development environment (auto-builds JS)
```
npm run dev
```

––> http://localhost:8080/

## Just run
```
npm start
```
